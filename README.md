## Features
- [Live music tracker](./src/lib/components/MusicPlaying.svelte).

  This component will show if I'm currently listening to music.
  
  I used

  - [Last.fm API](https://www.last.fm/api) to get the current track I'm listening.
  - [My LastFM Middleware](https://github.com/TomPlanche/LastFM-API-middleware) to get the current track I'm listening.
